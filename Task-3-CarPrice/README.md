#  Car Price Prediction using Machine Learning

##  Overview
This project predicts the selling price of used cars using machine learning algorithms based on features like year, fuel type, transmission, and kilometers driven.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

##  Workflow
- Data Collection
- Data Cleaning & Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation

##  Models Used
- Linear Regression
- Random Forest Regressor

##  Result
Random Forest model performed better than Linear Regression in predicting car prices.

##  Visualization
Actual vs Predicted price graph was plotted to evaluate model performance.

##  Conclusion
The model can effectively predict car resale prices and can be further improved using advanced algorithms.

##  Future Scope
- Use Deep Learning models
- Deploy as a web app
- Improve accuracy with more data
