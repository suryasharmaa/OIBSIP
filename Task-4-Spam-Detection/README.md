# Email Spam Detection using Machine Learning

##  Project Overview
This project is a Machine Learning model that classifies SMS messages as **Spam** or **Ham (Not Spam)**.

It helps in detecting unwanted or fraudulent messages automatically.


##  Dataset
- SMS Spam Collection Dataset
- Contains labeled messages:
  - Spam
  - Ham

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Natural Language Processing (NLP)
- 
##  Model Used
- Multinomial Naive Bayes

##  Steps involved
1. Data Cleaning
2. Text Preprocessing
3. Feature Extraction (CountVectorizer)
4. Train-Test Split
5. Model Training
6. Prediction
7. Accuracy Evaluation

## Result
- Model Accuracy: **~98%**
